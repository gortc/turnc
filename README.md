# TURNc

Package `turnc` implements TURN [[RFC5766](https://tools.ietf.org/html/rfc5766)] client.
Based on [pion/stun](https://github.com/pion/stun) and [gortc/turn](https://github.com/gortc/turn) packages.

*Work in progress*
